Filmsphere

Filmsphere is a React-based web application that allows users to browse a curated collection of movies, sign up, log in, and manage their profiles.

My Role & Tasks

In this project, I worked on both the frontend and backend. My tasks included designing the user interface, implementing user authentication using JWT, and developing API endpoints for movie browsing and profile management. I also handled the database integration with MongoDB Atlas.

Key Decisions & Consequences

A key decision was choosing React for the frontend and Node.js with Express for the backend. This allowed for fast development and smooth integration between the client and server sides. Using JWT for authentication ensured secure user access, while MongoDB Atlas provided a scalable database solution. These choices enabled the app to handle user authentication and manage user profiles efficiently.

What I Would Do Differently

If I were to redo this project, I would improve the movie browsing experience by adding a recommendation engine based on user preferences. Additionally, I would explore using a more sophisticated state management tool like Redux to handle complex data flows in the frontend.

Lessons Learned

This project reinforced my skills in full-stack development, particularly integrating React with a Node.js backend. I also learned valuable lessons in managing user data securely using JWT and handling RESTful API design.

Technologies Used

  • Frontend: React, Bootstrap
  • Backend: Node.js, Express
  • Database: MongoDB Atlas
  • Authentication: JWT (JSON Web Tokens)
  • Deployment: Heroku

API Endpoints

  • GET /movies: Get all movies
  • GET /movies/:Title: Get movie by title
  • GET /movies/genres/:genreName: Get movies by genre
  • GET /movies/directors/:directorName: Get movies by director
  • POST /users/:Username/movies/:MovieID: Add movie to favorites
  • DELETE /users/:Username/movies/:MovieID: Remove movie from favorites
