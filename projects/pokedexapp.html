Pokedex App

The Pokedex App is a simple web application that allows users to explore a list of Pokemon, view basic information, and search for specific Pokemon by name.

My Role & Tasks

In this project, I was the sole developer. My tasks involved creating the entire application, from setting up the basic HTML structure to handling API calls and building out the user interface with JavaScript. I also implemented the live search functionality, allowing users to filter Pokemon by name in real-time.

Key Decisions & Consequences

One key decision I made was to use Bootstrap for the front-end framework. This allowed me to quickly set up a responsive layout and ensure a consistent design. Another important choice was integrating the PokeAPI, which provided a robust and comprehensive dataset for the application. These decisions resulted in a streamlined development process and a user-friendly interface.

What I Would Do Differently

If I were to do this project again, I would focus on optimizing the performance of the API requests and potentially add more interactive features, such as the ability to compare Pokemon or add favorites. Additionally, I would explore adding more sophisticated error handling for failed API requests.

Lessons Learned

Through this project, I gained a deeper understanding of working with external APIs and improved my skills in managing asynchronous JavaScript. I also learned how important it is to plan and design a user interface that is intuitive and easy to use.
